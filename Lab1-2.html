Signing Up for GitHub

    How to create a GitHub account

  1. Open a web browser and go to github.com.
  2. Click on the sign up button in the top right corner.
  3. Fill out the registration form with your username, email address, and password.
  4. Click the create account button.
  5. Complete the CAPTCHA verification, if prompted.
  6. Choose the free plan.
  7. Click continue.
  8. Once the account is made. You have the option of adding a profile picture and some personal information.
  9. You can also follow the suggestions to personalize your experience.
  10. A link sent to your email. Follow the link to verify your email address.
  11. You now have a verified GitHub account.

    Adding Someone as a Collaborator to a Repository

  1. Navigate to the GitHub repository where you want to add a collaborator.
  2. Click on the settings tab at the top right of the repository page.
  3. Alternatively, there may be a box on the right side of the screen that allows you to invite collaborators.
  4. In the left sidebar, select collaborators under the manage access section.
  5. Click add people.
  6. Start typing the collaborator's GitHub username or email address in the search box.
  7. The collaborator's name should appear in the dropdown list.
  8. Choose the access level you wish for them to have (read, write, or admin.)
  9. Click the add collaborator button to invite the person.
  10. The collaborator should receive an email invitation. Once they accept, they will have access to the repository.

    How to Add HTML Files to Your Repository

  1. Navigate to your local repository directory using a terminal or command prompt.
  2. Place your HTML files in this directory.
  3. Open the terminal/command prompt and navigate to the repository directory.
  4. Use the following commands to add, commit, and push your HTML files to GitHub:
    • git add . (to stage your changes)
    • git commit -m "Add HTML files" (to commit your changes)
    • git push origin master (to push changes to GitHub)
