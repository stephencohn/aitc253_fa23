Lecture Notes For Class On 05SEPT2023

  • Today's class was on the basics of webpage designs. We talked about how to start the code and the site. One of the things talked about was how to add headers to the page. We do that by typing h followed a number 1-6 inside angle brackets, inserting your title, then closing it out with /h followed by the same number inside angle brackets.
  • Another thing we learned is how to make paragraphs. Creating a paragraph is similar to creating a header. Except we put p and /p inside the angle brackets.
  • We also learned that we can create bulletins. The process for that is the exact same as the paragraphs. However, instead of placing p and /p inside the angle brackets, we put li and /li.
  • On the lab handout, it shows that we can use ol or ul inside angle brackets to make lists. ol is an organized list, and ul is an unorganized list.

      • Let's put a summary of what was learned so far into an organized list such as this one.

    1. Type h 1-6 and close out with /h 1-6 inside angle brackets to make a header.
    2. Type p 1-6 and close out with /p 1-6 inside angle brackets to make a paragraph.
    3. Type li and close out with /li inside angle brackets to make a list.
    4. Use ol and ul to make organized or unorganized lists.

      Here is an unorganized list of random words.

    • Envelope
    • Flask
    • Computer
    • Peppermint

    As you can see, organized lists creates a list with numbers.
